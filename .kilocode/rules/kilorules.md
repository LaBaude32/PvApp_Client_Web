# kilorules.md

## Project Architecture

This is a VueJs application with :

- Components in `/src/components`
- Views in `/src/views`
- Composables in `/src/composables`
- Router using vue-router in `/src/router`
- Utilities in `/src/utilities`
- State management using Pinia in `/src/store`
- Vuetify for UI and components
- Axios for API calls

## Coding Standards

- Use VueJs composition API with script setup syntaxe
- Use TypeScript for all new files
- Follow the existing naming conventions
- Write tests for all new features
